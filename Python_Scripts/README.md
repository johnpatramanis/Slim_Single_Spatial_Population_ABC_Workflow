## Summary of what each python script does:
<br/><br/>


**```Find_Admixture_and_Assign_Trees_to_Pop.py```**: Very crucial script. This script takes as input the generated tree sequences from the Slim simulation. A user-defined number of samples are drawn from the last generation of the simulation (present day). For each of these sampled individuals, for each chromosome and for each copy of the chromosme, every tree is checked. The haplotype's tree is climbed upto the first generation of the Slim simulation. Depending on which population it coalesces to, the haplotype is then assigned an ancestry named after that population. E.g. if A segment that originates from 'population 1' is assigned 'ancestry 1'.
